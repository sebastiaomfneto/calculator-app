## Run Locally

Clone the project

```bash
  git clone https://github.com/sebastiaomfneto/spotify-client-pwa.git
```

Go to the project directory

```bash
  cd spotify-client-pwa
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

  
## Running Tests

To run tests, run the following command

```bash
  npm run test
```
